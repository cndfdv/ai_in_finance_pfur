# Контест — Home Credit Default Risk

В этой лабораторной работе необходимо принять участие в открытом соревновании Kaggle **Home Credit Default Risk** и построить модель для оценки кредитного риска заёмщика.

Подробное описание задачи, структура данных и правила отправки решений приведены на странице соревнования:

https://www.kaggle.com/competitions/home-credit-default-risk

## Задача

Необходимо предсказать вероятность дефолта заёмщика по кредитной заявке.

Тип задачи: **бинарная классификация**.

Целевая переменная: `TARGET`.

## Метрика

Качество решения оценивается по метрике **ROC-AUC**.

## Формат решения

Для каждого клиента из тестовой выборки необходимо предсказать вероятность дефолта в диапазоне от 0 до 1.

Решение загружается на Kaggle в формате `submission.csv`.

## Оценивание

| Оценка | Значение ROC-AUC |
|---:|---:|
| 5 | ≥ 0.790 |
| 4 | 0.770–0.790 |
| 3 | 0.750–0.770 |
| 2 | 0.700–0.750 |
| 1 | 0.600–0.700 |
| 0 | < 0.600 или отсутствие сабмишена |
