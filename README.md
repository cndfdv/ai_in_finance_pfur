# Искусственный интеллект в финансах

Курс для магистров, РУДН, Факультет искусственного интеллекта.

## Лабораторные работы

| Задание | Тема | Формат | Метрика |
---------|------|--------|---------|
| [Контест — Home Credit](lab_home_credit/) | Кредитные риски | Индивидуально | ROC-AUC |
| [Детекция мошенничества](lab_fraud/) | Фрод и аномалии | Индивидуально | ROC-AUC |
| [Прогноз волатильности по новостям](lab_news/) | NLP в финансах | Индивидуально | RMSE (abs_return) |

## Оценивание

Оценка от 0 до 5 по результату на Kaggle Public LB. Пороги:

| Оценка | Home Credit (ROC-AUC) | Fraud (ROC-AUC) | News (RMSE, abs_return) |
|---|---|---|---|
| 5 | ≥ 0.790 | ≥ 0.985 | ≤ 0.0064 |
| 4 | 0.770 – 0.790 | 0.950 – 0.985 | 0.0064 – 0.0065 |
| 3 | 0.750 – 0.770 | 0.900 – 0.950 | 0.0065 – 0.0067 |
| 2 | 0.700 – 0.750 | 0.800 – 0.900 | 0.0067 – 0.00720 |
| 1 | 0.600 – 0.700 | 0.600 – 0.800 | 0.00720 – 0.00800 |
| 0 | < 0.600 / нет сабмишена | < 0.600 / нет сабмишена | > 0.00800 / нет сабмишена |

Описание каждой задачи — в README соответствующей лабы.
